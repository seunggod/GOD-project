# GOD ( 여행 정보 공유 웹 사이트 )
## 프로젝트 개요
+ JSP를 이용한 여행 정보 공유 웹 사이트 구현 프로젝트
+ 3인 팀 프로젝트
+ 개발 기간 : 2021.11.30 ~ 2021.12.13 (총 14일)
+ DB 설계, 게시판, 검색, 페이징

## 주요 기능 설명
회원가입 기능

로그인 기능
+ 관리자 계정 로그인 시 각 페이지에 관리자 기능 활성화

게시판 기능
+ 추천순, 조회순, 날짜순 정렬 기능
+ 페이징, 검색 기능
+ 관리자 계정인 경우 모든 게시물 관리 가능

게시물 작성, 조회 기능
+ 이미지 업로드 가능
+ 추천, 댓글, 신고 기능
